# 📚 Book Donation Platform

Plataforma web para doação e troca de livros, conectando pessoas que desejam doar com quem precisa.

---

## 🚀 Funcionalidades

- Cadastro de livros
- Listagem de livros disponíveis
- Sistema de doação
- Interface simples e intuitiva

---

## 🛠️ Tecnologias

- React
- Vite
- JavaScript
- SCSS
